The ultimate goal of reporting is to add value to the product so that it can optimize processes and cut costs. The way to get to bring that added value of a path of uneven ground with mud and stones (raw data) to, following a series steps, become a paved towards improved decision making highway:

 

One could say that each of these phases are like dominoes, where if a chip fails it will be difficult or impossible to get add value to reach. We start with some data through reporting on information we convert that information by analyzing knowledge to support decision making. When these decisions are materialized in concrete actions it is when we have really provided value.

 

Operational reporting

It allows a study of the data at the day to day work so that they can monitor the daily performance of an area or organization. Usually it has three essential characteristics. In the end it is a support for making operational decisions. It provides real-time information. The information must be shown to the minute, hour, day or month, but always with this updated information. Detailed Information .- The information should present data on the highest level of detail possible

 

Analytical reporting

On the other hand, there is also the need to detect trends over the behavior of the data as well as an analysis over time. This is a support for strategic decision making. It provides near real-time information only based on historical data The info is often aggregated or summary It is embodied in the following: Listings areas: a series of key indicators for each area are detected and compared with historical data temporal done. Dashboard: It is a useful measure progress in operating results in an organization tool, but is insufficient to achieve the implementation of a Plan Objectives Estratégico.sus not come from the Strategy but the various plans and programs have to meet each area or department Dashboard or Scorecard: In this case is very similar to the previous tool, but in this case its purpose is to measure the success of alignment with the business plan. Its importance is that -through their indicators-sampled continuously when the Organization and its partners achieve the goals of the Strategic Plan. Scorecard or Balanced Scorecard: Its author Robert Kaplan calls it an administrative system that goes beyond the financial perspective that managers tend to evaluate the progress of a company. There are tools that facilitate the analysis stage to support better decision-making, these tools try to predict behavior and trends, and make it through data mining, trying to turn information into knowledge. There are two types of analysis:

 

Descriptive analysis

Given a set of data we may want to know if there are similarities between each of the samples. For this we have clustering and profiling tasks, both considered descriptive analysis tasks unsupervised (not determined the response generated) .It is important to remember that the result will not tell us which groups have in common, something that we have to play us so to create appropriate content to each set newsletters. The number of clusters is also a sore point and can set this value libremente.El profiling is another task of data mining makes a descriptive analysis of the samples. Very useful in the detection of fraud, for example. Given a data record with the variables that define the behavior of credit card users, we want to know how the sample is distributed and alert in case of get out of the pattern.

 

Predictive anaysis The analysis builds predictive models to infer the response of a sample (dependent variable) given a set of variables that define (independent variables). In most cases we use artificial intelligence algorithms, learning from historical data to those who do know the answer. Some of the most popular algorithms in this context are: decision trees, random forests, support vector machines (SVM), neural networks, boosting or linear regression. Algorithms are monitored and if determined response, unlike the clustering and profiling.
